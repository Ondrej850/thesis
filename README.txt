# Historical Cipher Generator

A professional Python desktop application for generating authentic 15th-century cipher documents with COCO format annotations.

## Installation

pip install -r requirements.txt

## Usage

python main.py

## Features

- Realistic paper aging effects
- Multiple cipher types (substitution, bigram, trigram, dictionary, nulls)
- COCO format annotations for ML training
- Interactive GUI with live preview
- Expandable SQLite database
